Hello World - Your first program

We are going to create a program named "Hello World". It is a program that displays this same message to the terminal. Nothing interesting I agree, but it is useful to understand the general concept of lim.

Set up the environment

Start by creating a new folder in which we will work. This step is not mandatory but strongly recommended.

Create a file named "hello_world.lim". The ".lim" extension is very important.

Programming

In the "hello_world.lim" file write the following code.

func main
    puts("Hello, World!")

The tabulation is important, the second line must be separated with a tabulation and not spaces.

Here we create a "main" function. A function is like a box in which we put code. You can call this box from almost anywhere to execute the code it contains. The "main" function is special because it is the one that is called at the very beginning of the execution of the program.

The following line belongs to the function because it is indented at least 1 time more than the line which declares the function (1st line). This line calls the "puts" function with the "Hello, World!" argument. The puts function writes the indicate argument to the terminal. Note that the text is in quotes to avoid any confusion with the keywords of the code.

Compilation

Compilation is a step that not all programming languages have. This step can be seen as a translation between the code you wrote and a binary file executable by your computer. This step is obviously automated by lim.

Open a terminal in the folder that contains your "hello_world.lim" file.

If you don't know how to use a terminal, we strongly advise you to watch an introductory video to this essential developer tool.

Run the following command

lim hello_world.lim hello_world

This command will then create a new file in your folder, this is the executable. This is independent of your code file, so you can take it wherever you want, it would still be functional.

Lim is used like this:

lim <your_code_file> [executable_name]

Execution

Now let's run our program in the terminal.

For windows

hello_world

For Linux

./hello_world

Output

Hello, World!

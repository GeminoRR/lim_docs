Str - Class

The str class represents a string.

Définition

primary class str

Properties

This class has no public property.

Constructor

This class cannot be defined via the "new" keyword.

Methods

int - To Int

This method parse the string to return integer (int).

Définition

func int:int

Arguments

This function takes no arguments.

Exemples

let age = gets("Enter your age : ").int()
let born_in = 2023 - age
puts(born_in.str())

Output (Assuming the user is entering 18) :

2005

Range - Function

The range function returns a list of integers in a certain field.

Arguments

fromMaximum number of the list, not included.

toMinimum of the list, includes

Return

list<int>The range function returns a list of increasing numbers between "from" and "to".

Exemples

This function is often used for for loops.

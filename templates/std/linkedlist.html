linkedlist - Linked List

The list class represents a linked list of elements.

This class is for use when you often want to change the position of list items.

Définition

class list<elm_type>

Properties

This class has no public property.

Constructor

func new:linkedlist<elm_type>

Methods

str - To String

This method returns the string representing the list.

Définition

func str:str

Arguments

This function takes no arguments.

Exemples

puts(["Hello", "World"].str())
["Hello", "World"]

len - Lenght

This method return the lenght (int) of the string.

Définition

func len:int

Arguments

This function takes no arguments.

Exemples

puts([True, True, False].len().str())
3

add - Add element

This method add a element to the end of the list.

Définition

func add(elm:elm_type)

Arguments

elmThe element to add at the end of the list.

Exemples

let cities = new linkedlist<str>
cities.add("Paris")
cities.add("New York")
cities.add("London")
puts(cities.str())
["Paris", "New York", "London"]

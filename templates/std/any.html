Any - Class

The "any" class allows you to contain a value regardless of its type. Beware though, to perform any action on the contained value, you first cast that value. Thus, only type comparisons are allowed.

Definition

class any

Properties

This function takes no arguments.

Constructor

This class cannot be defined via the "new" keyword.

Methods

str - To String

This method returns the string representing the number.

Définition

func str:str

Arguments

This function takes no arguments.

Exemples

puts(3.14.str())
puts(0.0001.str())
3.14
0.0001
